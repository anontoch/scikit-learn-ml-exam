# scikit-learn-ml-exam

Machine learning coursework grouped by topic.

## Groups

| Folder | Contents |
|---|---|
| `titanic_exam` | `titanic_preprocessing.py`, `titanic_dnn_pipeline.py`, phase scripts, `train.csv` / `test.csv` |
| `heart_disease` | `heart.csv`, `heartd.py`, `heart_desis.py` |
| `bank_customer_regression` | Bank customer ANN/LR, `dataset/BankCustomer.csv`, advertising regression |
| `customer_charts` | `customer_numeric_charts.py`, `customers-100.csv` |
| `linear_regression_demo` | sklearn linear regression demo |
| `regression_demos` | Salary, Boston housing, temperature, study-hours demos |
| `web_scraping` | `amazon_scraper_translate.py` |

## Run

Always `cd` into the script's folder first:

```powershell
cd titanic_exam
pip install pandas numpy scikit-learn tensorflow keras
python titanic_preprocessing.py
```

```powershell
cd bank_customer_regression
python bank_customer_ann_basic.py
```
